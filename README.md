# Pneumonia-Detection

Classification project

# Steps

Setup python environment

```
python3 -m venv aimi
source aimi/bin/activate
```

Make sure your venv's site-packages folder is the same python version as your kernel

Missing any libraries? (ModuleNotFoundError)

```
pip install
```
